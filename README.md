# Vikrey

> Проект основан на фреймворке Vue.js

## Установка

``` bash
# установка зависимостей
npm install

# запуск сервера с "горячей заменой" модулей на localhost:8080
npm run dev

# сборка для продакшена с минификацией файлов
npm run build
```
